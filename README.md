# 수면 무호흡증 분석 연구

## 📌 프로젝트 개요
- **설명:** 오픈소스 데이터셋을 활용하여 Bi-LSTM과 Dense 모델을 이용한 수면 무호흡증 분석 연구.
- **진행 기간:** 2024.10.01-2024.11.30
- **역할:** 
  - Apnea-ECG Database 활용
  - Bi-LSTM과 Dense 조합 모델 사용
  - 87%의 정확도를 가지는 모델 제안
  - TP, TN, FP, FN 등의 지표 계산
  - 실무보고서 우수상, 포스터 장려상 수상
